# 專案介紹
一個能查詢指定地點天氣並持續追縱的App
# 使用技術
**Vue3 + Vue-Router4 + pinia + Vite + tailwindCSS + axios** 
# 專案特色
1.中英文與攝氏華氏溫度切換  
2.使用tailwindCSS進行樣式開發，並以flex布局作RWD設計  
3.使用真實天氣api接口  
# 安裝依賴
### `npm install`
# 啟動專案
### `npm run dev`

# 使用方式
1.打開app並且搜尋想要追縱的城市  
2.選擇該城市查看天氣  
3.按+進行追縱
